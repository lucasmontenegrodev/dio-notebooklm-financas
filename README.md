# dio-notebooklm-financas
# Desafio DIO: Guia de Estudo Ativo com NotebookLM 

Este repositório contém o resultado do desafio da plataforma DIO, focado em utilizar o **Google NotebookLM** como uma ferramenta de aprendizagem ativa. O tema escolhido foi **Educação Financeira para Empreendedores e Profissionais Autônomos**.

##  Objetivos do Projeto
* Aplicar curadoria de fontes oficiais (Banco Central, CVM, SEBRAE).
* Estruturar um fluxo de estudo utilizando Engenharia de Prompts.
* Criar um material de consulta rápida (Miniguia) validado por IA.

##  Tecnologias e Ferramentas
* **[NotebookLM](https://notebooklm.google/)**: Para análise de documentos e geração de insights.
* **Markdown**: Para documentação e estruturação do conhecimento.
* **Git/GitHub**: Para versionamento e exposição do projeto.

##  Organização do Repositório
* `fontes/`: Documentos base em PDF.
* `prompts/`: Registro da evolução das perguntas feitas à IA.
* `guia-final/`: Resumo estruturado, glossário e dicas práticas.

##  Metodologia de QA (Qualidade)
Como foco em **Quality Assurance**, realizei a validação das respostas da IA cruzando as informações com as citações diretas das fontes, garantindo que não houvesse alucinações nos conceitos de taxas (Selic, IPCA) e regras de investimento.
